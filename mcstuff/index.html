Minecraft Stuff

A bunch of random things I made for Skript, a Minecraft plugin that allows you to code your own mini-plugins (they don't have to be mini) without needing a lot of knowledge of coding. There are also a few datapacks that I made.

Skripts

Moderation Tools

  Download

A skript with many moderation tools such as detecting blacklisted words, tempbanning, warning, and muting/tempmuting. You can customize all sorts of things such as all the blacklisted words, the amount of warns for a ban, and default ban/mute messages.
